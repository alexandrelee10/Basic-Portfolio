Alexandre Lee — Portfolio Website

A personal developer portfolio built to showcase my experience, skills, and projects as a Backend & Frontend Software Engineer. The site includes a hero section, about section, experience timeline, project gallery, and a fully functional contact form using EmailJS.

This portfolio highlights my ability to build clean, responsive interfaces while also showcasing my backend skills. It serves as a central place for recruiters to view my experience, projects, and contact information as I apply for software engineering roles.

Section Breakdown:

Hero Section - Introduces who I am, my role (Backend Software Engineer), and provides quick access to my portfolio or resume.
About Me Section - A short biography with skills and personal details. Contains animated progress bars showing proficiency in HTML, CSS, TypeScript, JavaScript, and React.
Experience Timeline - A vertical timeline showcasing previous roles (with placeholder text). Each section includes a logo, title, dates, and description.
Portfolio Projects - Card-style project gallery that displays project images, overlays, and titles. (Currently using placeholder images.)
Contact Section - A working contact form using EmailJS. Users can send me messages directly and receive a response in 24–48 hours.


Tech Stack:
- HTML5
- CSS3
- JavaScript
- EmailJS
- Ionicons
- Google Fonts
